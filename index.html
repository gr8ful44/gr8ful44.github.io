สวัสดีค่ะ ชื่อลภัสนะคะหรือเรียกหมูหยองก็ได้ จบปริญญาตรีด้านบริหารเอกการตลาดมาค่ะ ช่วงกลางปี2022ที่ผ่านมา เปิดอินเทอร์เน็ตเล่นไปเรื่อยเปื่อยจนได้มารู้จักกับวงการ(coding คนนอนน้อย)นี้โดยบังเอิญแล้วเกิดติดใจขึ้นมาค่ะ หลังจากนั้นก็เริ่มเรียนรู้ด้วยตัวเองผ่านช่องทางออนไลน์มาเรื่อยๆ ผิดบ้างถูกบ้าง พัฒนากันไป ตอนนี้สนใจการทำ UI/website/web page ที่เกี่ยวกับ E-Commerce เป็นพิเศษ เหตุผลคือ..ความชอบล้วนๆเลยค่ะ ชอบดีไซน์ ชอบขาย ชอบดูการพรีเซนต์ของเว็บไซต์ต่างๆที่เปิดเข้าไปแล้วมันว้าวจึ้ง เห็นแล้วอยากทำบ้างอะไรแบบนี้ พวก lib. ที่สนใจเป็นพิเศษตอนนี้ก็จะเป็น react ค่ะ ส่วนเป้าหมายในปัจจุบันคืออยากเรียนรู้เกี่ยวกับการทำเว็บไซต์ทางฝั่ง front-end ให้ได้มากที่สุด ใช้ tools ต่างๆให้คล่องแคล่วมากขึ้น เรียนรู้ library และ framework ที่มันเจ๋งๆ คติคืออยากเป็นเซียนฝั่งหน้าบ้านค่ะ
